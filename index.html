Customer Product Manager is a full-stack web application developed using React and Node.js that enables efficient management of customer information and product inventories. The platform allows users to add, update, and track customer data, link customers to products, and monitor sales or service interactions. Virtual Mouse

Virtual Mouse is a computer vision-based desktop application developed in Python, utilizing libraries such as OpenCV, MediaPipe, Tkinter, and Pandas. It allows users to control the mouse cursor using hand gestures captured via a webcam. The system detects hand landmarks in real-time and translates gestures into mouse actions like movement, clicks, and scrolling, providing a touchless, intuitive interface for hands-free interaction.
